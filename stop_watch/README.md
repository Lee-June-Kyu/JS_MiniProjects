# 간단한 스톱와치 만들기
< 자바스크립트 기초 내용 학습 >
<br/><br/>

## 주요 내용
1. setInteval과 clearInterval 응용해서 사용하기
> Date()생성자를 이용하지 않고 setInterval을 이용해서 시간의 흐름을 측정할 수 있다 <br/>
> clearInterval을 이용해서 시간을 멈춘것 처럼도 사용 할 수 있다
2. 적절한 템플릿 리터널 ( 백틱 ) 사용하기 
> 템플릿 리터널을 적절히 사용하면 매우 효율적이다 <br/>

<br/>

## 활용 가능성
- 간단한 시간 측정과 같은 경우에 매우 유용하게 사용가능하다 <br/>
- 시간 측정뿐 아니라 주기적인 반복 사용이 필요할때 setInterval을 사용 할 수 있다.
